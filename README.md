題目三、電子元件（CPU）冷卻速度預測與建模分析

熱傳導與對流——牛頓冷卻定律之應用
11215017 黃勝揚

---

## 1. 工程問題背景
隨著高性能計算（HPC）與電競產業發展，CPU 的功耗（TDP）日益增加。若散熱系統失靈，電子元件溫度過高將導致「降頻」甚至硬體毀損。
**核心問題：** 當 CPU 停止運作或散熱器啟動後，元件溫度隨時間下降的速率如何精確預測？

---

## 2. 物理建模：牛頓冷卻定律
根據牛頓冷卻定律，一個物體損失熱量的速率，正比於該物體與周圍環境的溫度差。

### 2.1 基本假設
1. **集總參數分析（Lumped Capacitance Model）**：假設 CPU 內部導熱極快（Biot Number $Bi < 0.1$），物體內部溫度均勻。
2. **環境溫度恆定**：冷卻過程期間，環境空氣溫度維持不變。

### 2.2 數學模型建立
設：
- $T(t)$：CPU 在時間 $t$ 的溫度
- $T_{\infty}$：環境溫度（室溫）
- $h$：對流換熱係數 (Heat Transfer Coefficient)
- $A$：散熱表面積
- $m$：物體質量
- $C_p$：比熱容

根據能量守恆，物體內能的減少率等於向環境散出的熱通量：
$$m C_p \frac{dT}{dt} = -h A (T - T_{\infty})$$

定義熱時間常數 $\tau = \frac{m C_p}{h A}$，則方程式可簡化為：
$$\frac{dT}{dt} = -\frac{1}{\tau} (T - T_{\infty})$$

---

## 3. 數學求解過程

我們需要解一個一階線性微分方程。

### 3.1 分離變數法
將變數 $T$ 與 $t$ 分離：
$$\frac{dT}{T - T_{\infty}} = -\frac{1}{\tau} dt$$

### 3.2 積分
兩邊同時積分（從初始時間 $t=0, T=T_0$ 到 任意時間 $t$）：
$$\int_{T_0}^{T(t)} \frac{1}{T - T_{\infty}} dT = \int_{0}^{t} -\frac{1}{\tau} dt$$

得到：
$$\ln \left( \frac{T(t) - T_{\infty}}{T_0 - T_{\infty}} \right) = -\frac{t}{\tau}$$

### 3.3 最終解
取指數函數 $\exp$ 得到 CPU 溫度隨時間變化的函數：
$$T(t) = T_{\infty} + (T_0 - T_{\infty}) e^{-\frac{t}{\tau}}$$

---

## 4. 案例模擬與數值分析



假設一個典型的 CPU 散熱場景：
* **初始溫度 $T_0$：** 80°C
* **環境溫度 $T_{\infty}$：** 25°C
* **時間常數 $\tau$：** 50 秒

| 時間 (s) | 預測溫度 (°C) | 說明 |
| -------- | ----------- | ---- |
| 0        | 80.0        | 剛停止高負載運作 |
| 50 ($\tau$)  | 45.2        | 溫度下降約 63.2% |
| 150 (3$\tau$) | 27.7        | 接近室溫 |

---

## 5. 結論
透過牛頓冷卻定律建模，我們可以發現：
1. **指數衰減**：冷卻速度在初期最快，隨著溫差縮小，冷卻速度逐漸放緩。
2. **優化方向**：若要加快冷卻速度（減小 $\tau$），工程上可透過**增加表面積 $A$**（加裝鰭片）或**提高對流係數 $h$**（增加風扇轉速）來達成。
